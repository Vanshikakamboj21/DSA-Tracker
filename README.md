# 📊 DSA Tracker

A simple and interactive **Data Structures and Algorithms (DSA) Tracker** built using HTML, CSS, and JavaScript.  
It helps users track solved questions, revise topics, and stay consistent in their DSA preparation.

---

## 🚀 Features

- Add DSA questions with topic name
- Mark questions as solved/unsolved
- Edit and delete questions
- Filter questions by topic (Stack, Tree, Graph, etc.)
- Dark/Light mode support
- Local storage support (data saved in browser)
- Simple and clean UI

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- Local Storage API
